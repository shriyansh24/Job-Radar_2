# Entrypoint: Research → Summarize → Decide Workflow

> Canonical skill: `workflows/research-summarize-decide`
> Canonical id: `workflows.research-summarize-decide`

## What this workflow does

- Explores a question or design space using the Researcher persona
- Synthesizes findings into a structured decision brief with options
- Produces a recommendation and concrete action plan

## When to use

- Evaluating libraries, APIs, or architectural approaches
- Investigating unfamiliar codebases before making changes
- Any decision that needs evidence before commitment

## When NOT to use

- Task is already well-understood and ready to implement
- Simple factual lookups

## Handoff

Provide: the question to answer, constraints, and acceptable uncertainty level.
