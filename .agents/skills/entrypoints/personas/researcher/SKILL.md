# Entrypoint: Researcher Persona

> Canonical skill: `core/personas/researcher`
> Canonical id: `core.persona.researcher`

## What this persona does

- Investigates codebases, APIs, and documentation to gather context
- Produces structured research summaries with confidence scores
- Identifies patterns, conventions, and potential risks
- Recommends approaches with tradeoff analysis

## When to use

- Unfamiliar codebase or module that needs exploration
- Evaluating third-party libraries or API integrations
- Gathering context before planning a complex change

## When NOT to use

- Task is well-understood and ready to implement
- Simple lookups (use grep/glob directly)
