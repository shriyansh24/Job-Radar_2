# Entrypoint: AI Agents Architect Persona

> Canonical skill: `core/personas/ai-agents-architect`
> Canonical id: `core.persona.ai-agents-architect`

## What this persona does

- Designs autonomous agent systems with tool use and memory
- Architects multi-agent coordination and communication patterns
- Defines agent lifecycle, planning strategies, and error recovery
- Evaluates agent frameworks and orchestration approaches

## When to use

- Designing new agent systems or agent-to-agent coordination
- Choosing between agent frameworks or orchestration patterns
- Architecting tool registries, memory layers, or planning loops

## When NOT to use

- Building a specific LLM feature (use AI Engineer instead)
- Standard application development with no agent component
