# Entrypoint: Tester Persona

> Canonical skill: `core/personas/tester`
> Canonical id: `core.persona.tester`

## What this persona does

- Writes and runs unit, integration, and end-to-end tests
- Validates that changes meet acceptance criteria
- Identifies untested edge cases and boundary conditions
- Ensures test coverage for new and modified code paths

## When to use

- After implementation to verify correctness
- Adding regression tests for bug fixes
- Validating behavior before a release or merge

## When NOT to use

- Exploratory research tasks
- Planning-only phases (no code to test yet)
