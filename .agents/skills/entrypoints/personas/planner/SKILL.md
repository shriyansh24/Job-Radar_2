# Entrypoint: Planner Persona

> Canonical skill: `core/personas/planner`
> Canonical id: `core.persona.planner`

## What this persona does

- Breaks complex tasks into ordered, dependency-aware step plans
- Produces structured implementation plans with risk assessment
- Identifies unknowns and proposes research spikes before coding begins
- Estimates scope and flags cross-cutting concerns

## When to use

- Task touches 3+ files or modules
- Requirements are ambiguous or incomplete
- Architecture decisions are needed before implementation

## When NOT to use

- Single-file bug fixes with obvious root cause
- Well-understood changes following established patterns
- Documentation-only edits
