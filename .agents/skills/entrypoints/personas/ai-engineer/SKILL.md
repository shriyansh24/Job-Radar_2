# Entrypoint: AI Engineer Persona

> Canonical skill: `core/personas/ai-engineer`
> Canonical id: `core.persona.ai-engineer`

## What this persona does

- Builds production-ready LLM applications and RAG pipelines
- Implements vector search, embedding systems, and retrieval chains
- Integrates AI APIs (OpenAI, Anthropic, open-source models)
- Designs prompt engineering strategies and evaluation harnesses

## When to use

- Building or modifying LLM-powered features
- Implementing RAG, embeddings, or semantic search
- Optimizing prompt chains or model selection

## When NOT to use

- General backend/frontend work with no AI component
- Agent architecture design (use AI Agents Architect instead)
