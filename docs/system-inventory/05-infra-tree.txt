| Path | Classification | Purpose |
|---|---|---|
| `D:/jobradar-v2/docker-compose.yml` | Operational source | Base service topology for local/container runtime |
| `D:/jobradar-v2/docker-compose.dev.yml` | Operational source | Dev overlay for hot reload and bind mounts |
| `D:/jobradar-v2/backend/Dockerfile` | Operational source | Backend container build and startup |

| `D:/jobradar-v2/.github/workflows/ci.yml` | Operational source | CI validation pipeline |
| `D:/jobradar-v2/.github/workflows/codeql.yml` | Operational source | Static security scanning |
| `D:/jobradar-v2/.github/workflows/dependency-review.yml` | Operational source | Dependency diff policy |
| `D:/jobradar-v2/.env.example` | Operational source | Intended env contract template |
| `D:/jobradar-v2/infra/redis/tls/.gitkeep` | Support placeholder | Empty placeholder, not active certificate configuration |
| `D:/jobradar-v2/README.md` | Support/admin | Human instructions, but not fully aligned with compose/config truth |
