# Frontend Inventory

Root: `D:/jobradar-v2/frontend`

## Frontend Architecture Summary

- Entry: `D:/jobradar-v2/frontend/src/main.tsx`
- Runtime root: `D:/jobradar-v2/frontend/src/App.tsx`
- Routing source of truth: `D:/jobradar-v2/frontend/src/App.tsx` plus `D:/jobradar-v2/frontend/src/lib/navigation.tsx`
- Theme source of truth: `D:/jobradar-v2/frontend/src/index.css` and `D:/jobradar-v2/frontend/src/store/useUIStore.ts`
- Server state: TanStack Query with page-local query keys
- Global client state: Zustand stores in `src/store`
- API boundary: axios wrappers in `src/api`
