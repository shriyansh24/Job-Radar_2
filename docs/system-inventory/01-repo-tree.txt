## Notes On Source Of Truth Vs Support

- The backend has no separate `db/` or `repositories/` directory. Persistent data contracts are spread across:
  - `backend/app/*/models.py`
  - `backend/app/migrations`
  - service-layer queries inside `backend/app/*/service.py`
- The frontend design system is source-controlled in code, but `frontend/system.md` is the design-language reference that explains the intended token/layout grammar.
- `docs/current-state` is useful context, but implementation truth still lives in `frontend/src`, `backend/app`, migrations, and runtime config files.
